## 🚀 About the Project
"Ethiopia in 60 Seconds" is a Next.js application designed to provide quick, immersive stories about Ethiopia's diverse regions. Users can explore food, music, clothing, and history through a modern, responsive interface.

## 🛠️ Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **UI Components:** shadcn/ui
- **Package Manager:** pnpm

## ✨ Key Features
- **Dynamic Regions:** Explore 13 different regions of Ethiopia.
- **Cultural Insights:** Categorized views for Food, Music, Heritage, and more.
- **Responsive Design:** Optimized for both mobile and desktop users.
- **Random Explorer:** A "Random Region" feature for quick discovery.
